# Example for .xy data

This is an example for the XY support from [SpecsLabProdigy](https://www.specs-group.com/nc/specs/products/detail/prodigy/). The example conversion can be run with the following command.

```sh
! dataconverter \
MgFe2O4.xy \
eln_data.yaml \
--reader xps \
--nxdl NXmpes \
--output MgFe2O4.nxs \
```
